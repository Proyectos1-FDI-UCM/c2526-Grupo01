Coulro es un videojuego de plataformas 2D en el que el jugador encarna a un bufón de juguete que cobra vida dentro de una juguetería. El objetivo principal es avanzar a través de distintas estancias de una juguetería evitando hacer ruido, ya que cualquier exceso puede alertar a los niños y provocar que el jugador sea descubierto.

El juego combina mecánicas clásicas de plataformas —como saltos, desplazamiento lateral y uso del entorno— con un sistema de vida basado en el control del ruido. El jugador deberá a recorrer la juguetería evitando los numerosos juguetes ruidosos que puedan delatar al protagonista.

A lo largo del nivel, el jugador encontrará distintos elementos interactivos, obstáculos y situaciones que pondrán a prueba su habilidad. La experiencia busca equilibrar habilidad y tensión, obligando al jugador a avanzar con cuidado mientras explora el entorno.


*CONTROLES:*

- Andar (A-D/Joystick)
- Saltar (SPACE/A)
- Sprintar (LeftSHIFT/RT)
- Dash (J/LT)
- Pausa (Esc/START)
- Interactuar/agarrar (E/Y)

*KNOWN ISSUES*

- El sistema de knockback no funciona correctamente en determinadas situaciones y puede generar comportamientos inconsistentes.

- Existe un problema con los checkpoints al completar el juego, aunque se trata de un error conocido, pero que sabemos solucionar.

- El comportamiento del robot puede ser inesperado, ya que fue diseñado para una versión anterior del sistema de movimiento (previa al uso de Rigidbody2D).

- La cámara presenta ligeros temblores durante toda la ejecución del juego.
